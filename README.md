# GymFlow

GymFlow es una aplicación web interactiva diseñada para organizar rutinas y tareas de entrenamiento de forma sencilla. Permite añadir y eliminar tareas y mantiene la información guardada incluso después de recargar la página gracias al uso de LocalStorage.

## Funcionalidades

- Añadir nuevas tareas de entrenamiento
- Eliminar tareas de la lista
- Persistencia de datos con LocalStorage
- Interfaz responsive adaptable a diferentes dispositivos
- Filtro de búsqueda para encontrar tareas rápidamente

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- LocalStorage
- Vercel para el despliegue de la aplicación

## Cómo usar la aplicación

1. Escribe una tarea en el campo de entrada.
2. Pulsa el botón **Añadir**.
3. La tarea aparecerá en la lista.
4. Puedes eliminar cualquier tarea pulsando **Eliminar**.
5. Las tareas se guardan automáticamente en el navegador.
