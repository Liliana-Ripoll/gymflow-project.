# Flujo de trabajo con Cursor

En este documento se explica cómo se ha utilizado Cursor como IDE asistido por inteligencia artificial 
durante el desarrollo del proyecto, detallando las tareas realizadas y cómo ayudó a mejorar el proceso de programación.

## Primer contacto con Cursor
Para esta fase se instaló Cursor y se abrió el proyecto GymFlow dentro del editor.

### Exploración de la interfaz
Se exploraron las diferentes partes del editor, como el explorador de archivos, la terminal integrada y el chat con inteligencia artificial que permite interactuar directamente con el código.

### Autocompletado con IA
Se probó el autocompletado escribiendo comentarios que describían funciones. Cursor fue capaz de generar sugerencias de código automáticamente a partir de esas descripciones.

### Uso del chat contextual
También se utilizó el chat integrado para pedir explicaciones sobre partes del código y entender mejor algunas funciones del proyecto.

### Edición de código
Cursor permitió modificar y mejorar funciones existentes utilizando sugerencias generadas por inteligencia artificial.

### Conclusión
Después de probar Cursor, se pudo comprobar que es una herramienta útil para ayudar durante el desarrollo, ya que facilita la generación de código, la comprensión de funciones y la mejora del proyecto.


### Atajos de teclado utilizados
Durante el uso de Cursor se utilizaron algunos atajos de teclado que facilitan el trabajo durante la programación:

- **Ctrl + P**: Buscar archivos dentro del proyecto rápidamente.
- **Ctrl + Shift + P**: Abrir la paleta de comandos.
- **Ctrl + /**: Comentar o descomentar líneas de código.

Estos atajos permiten trabajar de forma más rápida dentro del editor.
