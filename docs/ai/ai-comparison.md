# Comparación de herramientas de IA

En este documento se comparan distintas herramientas de inteligencia artificial utilizadas durante el desarrollo del proyecto, analizando sus 
funciones, ventajas, diferencias y utilidad práctica en el flujo de trabajo.

## Herramientas de IA utilizadas

Durante el desarrollo del proyecto se han utilizado diferentes herramientas de inteligencia artificial para mejorar el proceso de programación y resolver problemas de forma más rápida.

### ChatGPT
ChatGPT se utilizó como asistente principal para resolver dudas sobre código, generar ejemplos de HTML, CSS y JavaScript, y mejorar partes del proyecto. También ayudó a entender errores y a optimizar la estructura del código.

### Claude
Claude se probó como alternativa a ChatGPT para comparar respuestas y obtener diferentes enfoques para resolver problemas de programación.

### Cursor
Cursor se utilizó como IDE asistido por inteligencia artificial. Permite generar código automáticamente, mejorar funciones y navegar por el proyecto con ayuda de IA.

### GitHub Copilot
GitHub Copilot funciona como un asistente de autocompletado que sugiere código mientras se programa, lo que permite desarrollar más rápido.


## Comparación entre ChatGPT y Claude

Para realizar una pequeña comparativa entre asistentes de inteligencia artificial, se pidió a ChatGPT y Claude que explicaran algunos conceptos técnicos de JavaScript, detectaran errores en funciones y generaran código a partir de descripciones.

### Explicación de conceptos

Se pidió a ambos asistentes que explicaran conceptos como el DOM, el event loop y las closures.

Ambos ofrecieron explicaciones correctas y fáciles de entender. En algunos casos ChatGPT utilizó ejemplos más directos, mientras que Claude ofreció explicaciones algo más detalladas.

### Detección de errores en código

También se crearon algunas funciones de JavaScript con errores intencionados para comprobar si los asistentes eran capaces de detectar el problema.

Los dos asistentes fueron capaces de identificar los errores y explicar qué estaba ocurriendo en el código, además de sugerir posibles soluciones.

### Generación de funciones

Por último, se describieron varias funciones en lenguaje natural para que los asistentes generaran el código correspondiente.

Tanto ChatGPT como Claude generaron código funcional y correcto, mostrando que ambos pueden ser útiles para ayudar en tareas de programación.

### Conclusión

Después de realizar estas pruebas, se puede concluir que ambos asistentes pueden servir como apoyo durante el desarrollo de software, ayudando a entender conceptos, detectar errores y generar código de forma más rápida.
