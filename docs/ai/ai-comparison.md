# Comparación de herramientas de IA

En este documento se comparan distintas herramientas de inteligencia artificial utilizadas durante el desarrollo del proyecto, analizando sus 
funciones, ventajas, diferencias y utilidad práctica en el flujo de trabajo.

## Herramientas de IA utilizadas

Durante el desarrollo del proyecto se han utilizado diferentes herramientas de inteligencia artificial para mejorar el proceso de programación y resolver problemas de forma más rápida.

### ChatGPT
ChatGPT se utilizó como asistente principal para resolver dudas sobre código, generar ejemplos de HTML, CSS y JavaScript, y mejorar partes del proyecto. También ayudó a entender errores y a optimizar la estructura del código.

### Claude
Claude se probó como alternativa a ChatGPT para comparar respuestas y obtener diferentes enfoques para resolver problemas de programación.

### Cursor
Cursor se utilizó como IDE asistido por inteligencia artificial. Permite generar código automáticamente, mejorar funciones y navegar por el proyecto con ayuda de IA.

### GitHub Copilot
GitHub Copilot funciona como un asistente de autocompletado que sugiere código mientras se programa, lo que permite desarrollar más rápido.
