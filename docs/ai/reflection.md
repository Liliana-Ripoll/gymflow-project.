# Reflexión final

En este documento se presenta una reflexión sobre el uso de herramientas de inteligencia artificial en el desarrollo del proyecto, valorando su utilidad,
los problemas encontrados y el aprendizaje obtenido durante el proceso.

## Reflexión sobre el uso de IA

Durante el desarrollo de este proyecto he podido probar cómo las herramientas de inteligencia artificial pueden ayudar bastante a la hora de programar. En mi caso, utilizar IA me ha servido para resolver dudas más rápido, entender mejor algunos errores del código y también obtener ideas para mejorar el diseño de la aplicación.

En varios momentos del proyecto utilicé la IA para revisar partes del código o para preguntarle cómo podía mejorar algunas secciones. Esto me ayudó a organizar mejor el código y a entender nuevas formas de hacer ciertas cosas.

También me di cuenta de que la IA puede ahorrar bastante tiempo cuando estás programando, sobre todo cuando tienes dudas o cuando necesitas una idea rápida para seguir avanzando con el proyecto.

Aun así, creo que es importante revisar siempre lo que genera la IA y entender bien el código que se utiliza, porque no siempre todo funciona perfecto a la primera.

En general, esta experiencia me ha ayudado a ver cómo la inteligencia artificial puede ser una herramienta muy útil dentro del desarrollo de software y cómo puede facilitar bastante el trabajo cuando se usa de forma correcta.
