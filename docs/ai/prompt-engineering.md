# Prompt Engineering

En este documento se recopilan los prompts utilizados durante el desarrollo del proyecto para interactuar con herramientas de inteligencia artificial como ChatGPT.
Se analizan los resultados obtenidos y cómo se pueden mejorar las instrucciones para obtener respuestas más precisas.

## Uso de prompts durante el desarrollo

Durante el desarrollo del proyecto se utilizaron distintos prompts para obtener ayuda de herramientas de inteligencia artificial como ChatGPT.

### Ejemplo 1: Mejora del diseño
Prompt utilizado:
"Crea un diseño responsive para una aplicación fitness utilizando HTML, Tailwind CSS y una estructura con header, sidebar y sección principal."

Resultado:
La IA proporcionó una estructura clara para organizar la interfaz de la aplicación, facilitando la creación de un diseño moderno y adaptable a diferentes dispositivos.

### Ejemplo 2: Solución de errores
Prompt utilizado:
"Revisa este código HTML y CSS y dime por qué el diseño no se ve correctamente en móvil."

Resultado:
La IA ayudó a identificar problemas en el diseño responsive y sugirió mejoras utilizando clases de Tailwind y media queries.

### Ejemplo 3: Optimización del código
Prompt utilizado:
"Optimiza este código JavaScript para hacerlo más limpio y fácil de mantener."

Resultado:
Se obtuvieron sugerencias para simplificar funciones y mejorar la organización del código.
