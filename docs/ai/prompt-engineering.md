# Prompt Engineering

En este documento se recopilan los prompts utilizados durante el desarrollo del proyecto para interactuar con herramientas de inteligencia artificial como ChatGPT.
Se analizan los resultados obtenidos y cómo se pueden mejorar las instrucciones para obtener respuestas más precisas.

## Uso de prompts durante el desarrollo

Durante el desarrollo del proyecto se han utilizado distintos prompts para interactuar con herramientas de inteligencia artificial como ChatGPT. Gracias a esto ha sido posible resolver dudas, mejorar el diseño de la aplicación y optimizar partes del código de forma más rápida.

### Ejemplo 1: Mejora del diseño
Prompt utilizado:
"Crea un diseño responsive para una aplicación fitness utilizando HTML, Tailwind CSS y una estructura con header, sidebar y sección principal."

Resultado:
La IA ayudó a generar una estructura clara para organizar la interfaz de la aplicación. Esto facilitó crear un diseño más moderno y adaptable a diferentes tamaños de pantalla.

### Ejemplo 2: Solución de errores
Prompt utilizado:
"Revisa este código HTML y CSS y dime por qué el diseño no se ve correctamente en móvil."

Resultado:
La IA ayudó a detectar algunos problemas en el diseño responsive y sugirió mejoras utilizando clases de Tailwind y ajustes en la estructura del código.

### Ejemplo 3: Mejora del código
Prompt utilizado:
"Optimiza este código JavaScript para que sea más limpio y fácil de entender."

Resultado:
Se obtuvieron sugerencias para simplificar algunas partes del código y hacerlo más organizado, lo que facilita su mantenimiento y comprensión.


# Prompts
## Prompt 1 – Explicar código a nivel básico
Prompt: Explica este código JavaScript de forma sencilla como si fuera para alguien que está aprendiendo programación.
¿Por qué funciona bien?: Ayuda a entender mejor el código del proyecto y facilita el aprendizaje.

## Prompt 2 – Generar ejemplos
Prompt: Dame tres ejemplos de cómo se podría mejorar la función addTask del proyecto GymFlow.
¿Por qué funciona bien?: Pedir ejemplos ayuda a obtener varias ideas o soluciones posibles.

## Prompt 3 – Definir un rol
Prompt: Actúa como un desarrollador frontend senior. Analiza el archivo app.js y sugiere mejoras para que el código sea más limpio y fácil de mantener.
¿Por qué funciona bien?: Definir un rol hace que la IA responda desde la perspectiva de un desarrollador experto, generando recomendaciones más profesionales.

## Prompt 4 – Detectar errores
Prompt: Revisa el archivo app.js y detecta posibles errores en el código.
¿Por qué funciona bien?: Permite usar la IA como herramienta de revisión de código.

## Prompt 5 – Generar comentarios
Prompt: Añade comentarios al código para explicar qué hace cada parte.
¿Por qué funciona bien?: La IA puede ayudar a documentar el código para que sea más fácil de mantener.

## Prompt 6 – Comparar soluciones
Prompt: Sugiere dos formas diferentes de implementar esta funcionalidad y explica cuál sería mejor.
¿Por qué funciona bien?: Comparar soluciones ayuda a tomar mejores decisiones de programación.

## Prompt 7 – Simplificar código
Prompt: Simplifica este código JavaScript manteniendo el mismo resultado.
¿Por qué funciona bien?: Ayuda a reducir código innecesario o repetido.

## Prompt 8 – Mejorar interfaz
Prompt: Sugiere mejoras visuales para la interfaz de la aplicación usando Tailwind CSS.
¿Por qué funciona bien?: Permite mejorar el diseño de la aplicación.

## Prompt 9 – Explicar mejoras posibles
Prompt: Analiza este código y sugiere tres mejoras que podrían hacerlo más eficiente.
¿Por qué funciona bien?: La IA puede detectar oportunidades para optimizar el código y mejorar su rendimiento.

## Prompt 10 – Proponer mejoras futuras
Prompt: Sugiere tres mejoras o nuevas funciones que podrían añadirse a la aplicación GymFlow en el futuro.
¿Por qué funciona bien? Permite usar la IA para generar ideas que ayuden a mejorar el proyecto a largo plazo.
